Sites worked on

Simple tools made

These are laughably simple tools, mostly created on whim. Every one of them is built with vanilla JavaScript, and some part of each is either incomplete or outdated.

Quip

A webpage with a calendar on it for doing some quick event planning. Short events can be added for the current month and seen in a sorted manner. Since all data is stored using JavaScript's "sessionStorage", everything is temporary and is lost on tab reload.

Qlick

A browser bookmarklet (webpage provided as well) to remove tracking queries from URLs. This was originally developed to prune URLs on Facebook. Later some support for Gmail and YouTube links were also added.

Disfigure

A browser bookmarklet to remove specific parts of a webpage to mimic a somewhat distraction-free interface. Only Facebook and YouTube on desktop browsers are supported.

Currently broken on Facebook due to their last UI change.

CS3

A webpage that sorts CSS declarations alphabetically. Both "pretty" and minified CSS are supported, though with limited functionalities.

Not out of development phase yet.

Rasputin

A webpage to generate random sequences of characters. Originally created for generating temporary online meeting links and passwords.

Not out of development phase yet.
